## 프로젝트 소개
좋아하는 Kpop 노래를 소개하는 웹사이트입니다.  
사용자가 앨범 표지를 클릭하면 노래 정보를 볼 수 있고, 재생 버튼을 누르면 유튜브로 이동합니다.

## 프로젝트 목적
각자 가장 좋아하는 K-Pop 그룹과 추천 곡 정보를 한 곳에 모아, 시각적이고 동적으로 소개하는 작은 웹 페이지를 제작하는 것이 목표입니다. 이는 단순한 정보 공유를 넘어, 우리의 취향을 담은 개인화된 K-Pop 아카이브를 구축하는 데 중점을 둡니다.
